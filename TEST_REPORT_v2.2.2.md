## Bug 修复记录

### Bug #1: 策略引擎空参数处理

**问题描述**: 当 `tool_args` 为 `None` 时，`evaluate_tool` 方法会抛出异常。

**修复方案**: 在 `evaluate_tool` 方法中添加空值检查。

**修复代码**:
```python
def evaluate_tool(self, tool_name: str, tool_args: Dict) -> PolicyResult:
    """快捷方法：评估工具调用"""
    if tool_args is None:
        tool_args = {}
    ctx = PolicyContext(
        tool_name=tool_name,
        tool_args=tool_args,
        command=tool_args.get("command", ""),
        file_path=tool_args.get("path", ""),
        content=tool_args.get("content", ""),
    )
    return self.evaluate(ctx)
```

**修复状态**: ✅ 已修复并验证
