**架构设计**
```
用户请求 → 策略引擎 → 模型路由 → 推理引擎 → 对抗验证 → 输出
              ↓              ↓           ↓           ↓
           规则库         模型池      记忆系统      工具集
```

- 🧠 **推理引擎**：第一性原理拆解、决策矩阵量化、根因分析

- 💾 **记忆系统**：双层架构（事实+摘要）、相关性检索
- 🔧 **工具编排**：terminal、文件、搜索、git、HTTP、Python执行
- 📊 **策略引擎**：可组合、可继承、可定制的规则系统

**技术亮点**

```python
# 多模型智能路由
class ModelRouter:
    def route(self, task: Task) -> Model:
```
